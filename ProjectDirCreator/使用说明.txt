# 项目目录创建工具使用说明

## 软件功能
本工具用于根据文本文件中的缩进结构自动创建项目目录结构，支持以下特性：
- 基于文本文件的缩进层次自动生成目录结构
- 支持空格和Tab混合缩进（自动转换Tab为4个空格）
- 支持跳过空行，提高文件可读性
- 图形界面操作，简单直观

## 安装说明
无需安装，直接运行即可：
1. 解压 `ProjectDirCreator.zip` 到任意位置
2. 打开 `ProjectDirCreator` 文件夹
3. 双击 `mk_proj_dir.exe` 运行程序

## 使用步骤
1. **选择树文件**：点击"选择文件"按钮，选择包含目录结构的文本文件（默认提供 `tree.txt` 作为示例）
2. **选择根目录**：点击"选择目录"按钮，选择要创建目录的目标位置
3. **创建目录**：点击"创建目录"按钮，程序会根据树文件中的结构创建对应目录
4. **完成**：看到"目录创建成功！"提示后，操作完成

## 树文件格式说明
树文件使用缩进表示目录层级关系，示例：
```
项目名称
    图纸
        参照
        说明
    资料
        规范
        项目信息
    发出
```
- 每行表示一个目录或文件
- 使用空格或Tab缩进表示层级（Tab会自动转换为4个空格）
- 空行会被自动跳过

## 示例 `tree.txt`
本工具默认包含一个示例 `tree.txt` 文件，内容如下：
```
1-设计图纸
    xr建筑
	xr结构
	other
2-说明计算
3-归档
4-提资
    弱电
    强电
    建筑
	结构
5-收资
    暖通
    结构
    建筑
6-外部配合
7-审图
8-施工配合
9-甲方资料
10-设计资料
11-邮件会议
```

## 注意事项
1. 确保目标根目录存在且具有写入权限
2. 树文件中的目录名应避免包含特殊字符
3. 程序会自动创建不存在的父目录
4. 已存在的目录不会被覆盖

## 系统要求
- Windows 10 及以上操作系统
- 无需安装 Python 或其他依赖

## 故障排除
如果遇到问题：
1. 检查树文件格式是否正确
2. 确认目标目录是否有写入权限
3. 尝试以管理员身份运行程序

# 项目目录创建工具

一款基于 tkinter 的桌面工具，通过定义树形文本结构快速创建项目目录。

## 新手入门（图解教程）

### 这个工具是做什么的？

简单来说，这个工具可以帮你**一次性创建多个文件夹**。

比如你要建立这样的文件夹结构：
```
我的项目
    ├── 图片
    ├── 文档
    └── 子文件夹
        ├── 视频
        └── 音频
```

用手工操作需要创建 5 次，而这个工具只需要写一段文字，一次搞定。

### 第一步：准备工作

1. 新建一个文件夹，命名为你喜欢的名字（比如"我的项目"）
2. 在这个文件夹里新建一个**文本文件**（右键 → 新建 → 文本文档），命名为 `tree.txt`
3. 用记事本打开 `tree.txt`，按照下面的格式输入内容

### 第二步：编写目录结构文本

打开 `tree.txt`，输入以下内容：

```
我的项目
    图片
    文档
    子文件夹
        视频
        音频
```

**关键点：**
- 每行写一个文件夹名字
- 子文件夹前面加 **4 个空格**（按 4 次空格键）
- 可以嵌套多层，只要多加空格即可

### 第三步：运行程序

1. 双击 `dist/mk_proj_dir.exe` 打开程序
2. 点击左上角"浏览"按钮
3. 选择你刚才创建的 `tree.txt` 文件
4. 点击中间偏下的"浏览"按钮
5. 选择你要创建文件夹的位置（比如 D 盘或桌面）

### 第四步：预览并创建

1. 观察右侧的预览区域，确认文件夹结构正确
2. 点击绿色的"创建目录结构"按钮
3. 完成！去你选择的目录看看，文件夹都已经创建好了

### 格式说明

| 你写的格式 | 效果 |
|-----------|------|
| `我的项目` | 根文件夹 |
| `    图片` | 根文件夹下的"图片"子文件夹（4个空格） |
| `        视频` | 图片文件夹下的"视频"子文件夹（8个空格） |

### 缩进示例

```
项目名称
    第一级文件夹A
        第二级文件夹A1
        第二级文件夹A2
    第一级文件夹B
        第二级文件夹B1
```

表示的文件夹结构：
```
项目名称/
├── 第一级文件夹A/
│   ├── 第二级文件夹A1/
│   └── 第二级文件夹A2/
└── 第一级文件夹B/
    └── 第二级文件夹B1/
```

## 功能特性

- **可视化预览**：实时预览目录结构，所见即所得
- **树形文本解析**：使用缩进（空格或 Tab）表示目录层级
- **批量创建**：一键根据模板创建完整目录结构
- **图形界面**：简洁现代的 UI 设计，操作直观
- **自动转换**：Tab 字符自动转换为 4 个空格

## 安装使用

### 环境要求

- Python 3.8+

### 运行程序

双击运行 `dist/mk_proj_dir.exe` 即可启动程序。

如果提示"无法运行"或缺少依赖，请先安装 Python：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8 或更高版本
3. 安装时勾选"Add Python to PATH"
4. 安装完成后，双击 `mk_proj_dir.exe` 运行

### 打包为可执行文件

```bash
pyinstaller mk_proj_dir.spec
```

打包后的可执行文件位于 `dist/mk_proj_dir.exe`。

## 树文件格式

树文件采用缩进格式表示目录层级关系，支持空格或 Tab 缩进。

### 示例

```
项目根目录
    源代码
        utils
        models
        views
    资源文件
        images
        styles
    文档
        设计文档
        使用指南
    tests
```

### 格式说明

| 元素 | 说明 |
|------|------|
| 顶层名称 | 根目录下的一级子目录 |
| 缩进 | 使用空格或 Tab 表示层级关系 |
| 空行 | 自动忽略 |
| Tab | 自动转换为 4 个空格 |

## 操作指南

### 基本操作步骤

1. **选择树文件**：点击"浏览"按钮，选择定义目录结构的 `.txt` 文件
2. **选择根目录**：点击"浏览"按钮，选择创建目录的目标位置
3. **预览结构**：确认右侧预览面板中的目录结构是否符合预期
4. **创建目录**：点击"创建目录结构"按钮生成目录

### 注意事项

- 树文件默认查找同目录下的 `tree.txt`
- 已存在的目录不会重复创建
- 程序启动后会自动加载同目录下的 `tree.txt` 并预览

## 核心类说明

### `Node`

表示目录树中的单个节点。

| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | `str` | 节点名称 |
| `level` | `int` | 节点层级（根为 -1） |
| `children` | `List[Node]` | 子节点列表 |
| `full_path` | `Path` | 完整路径 |

### `Tree`

负责解析树文件和管理目录结构。

| 方法 | 说明 |
|------|------|
| `mk_tree_from_txt(path, root)` | 从文本文件解析目录结构 |
| `mk_dir_from_tree(root, tree)` | 根据树结构创建目录 |

### `ProjectDirCreatorGUI`

图形界面主类，提供完整的用户交互功能。

## 常见问题

**Q: 缩进使用空格还是 Tab？**
A: 两者都支持，Tab 会自动转换为 4 个空格。

**Q: 如何创建嵌套多层的目录？**
A: 通过增加缩进层级来表示，例如：
```
根目录
    第一层
        第二层
            第三层
```

**Q: 目录已存在会怎样？**
A: 使用 `mkdir(parents=True, exist_ok=True)`，已存在的目录不会报错。
